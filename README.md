# point-n-click

Prototype for college project in [LÖVE](https://love2d.org).

## Running from source

1. Clone source<br>
```bash
git clone https://github.com/Calebri/point-n-click.git
```
2. Move to repo directory<br>
```bash
cd point-n-click
```
3. Run LÖVE in repo directory<br>
```bash
love .
```
